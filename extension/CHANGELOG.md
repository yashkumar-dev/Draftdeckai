# Changelog

All notable changes to the DraftDeckAI Interview & DSA Prep Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-07

### Added
- 🧩 **DSA Problem Solver**
  - AI-powered solutions for coding problems
  - Support for JavaScript, Python, Java, C++, C#
  - Step-by-step approach explanations
  - Time and space complexity analysis
  - Multiple AI provider support (Gemini, OpenAI, Claude, Mistral)

- 💼 **Interview Preparation**
  - Role-specific question generation
  - Experience level tailoring (Entry, Mid, Senior)
  - Company-specific preparation
  - Practice mode with show/hide answers
  - Technical, behavioral, and system design questions

- 📄 **Career Tools**
  - AI-powered resume review
  - Cover letter generator
  - LinkedIn profile optimization
  - Salary guide and negotiation tips

- 🎮 **Gamification System**
  - XP and level progression (Beginner to Legend)
  - Achievement system with unlockable badges
  - Streak tracking for daily practice
  - Progress dashboard with statistics

- 🎤 **Voice & Interview Mode**
  - Voice command support
  - AI-powered mock interview simulator
  - Real-time answer evaluation
  - Comprehensive feedback system

- 🌐 **Platform Integration**
  - Auto-detection for LeetCode, HackerRank, Codeforces, GeeksforGeeks
  - Floating helper button on problem pages
  - Context menu integration (right-click to solve)
  - Smart problem extraction

- 🔧 **Core Features**
  - Multi-AI provider configuration
  - Dark mode support
  - Secure local storage
  - Privacy-focused (no data collection)
  - Chrome/Edge Manifest V3 support

- 📚 **Documentation**
  - Comprehensive README with installation guide
  - API key setup instructions
  - Troubleshooting section
  - Contributing guidelines

### Technical Details
- Manifest Version: 3
- Background Service Worker architecture
- Content Scripts for platform integration
- Modern ES6+ JavaScript
- Responsive UI design
- Cross-platform compatibility

### Supported Platforms
- LeetCode
- HackerRank
- Codeforces
- GeeksforGeeks
- LinkedIn (profile optimization)
- Indeed (job search)
- Wellfound/AngelList (startup jobs)
- Glassdoor (company research)

### Supported AI Providers
- Google Gemini (recommended, free tier available)
- OpenAI (GPT-4o-mini and GPT-4)
- Anthropic Claude (Claude 3 Haiku)
- Mistral AI (Mistral Small)

### Security
- No API keys hardcoded
- Secure browser storage
- HTTPS-only API communications
- Content Security Policy enforced
- No external data collection

### Known Issues
- None at initial release
